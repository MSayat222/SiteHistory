## üéØ Overview

This educational website presents a comprehensive analysis of how Russian colonial administrative codes transformed Kazakh society during the 18th and 19th centuries. Based on archival materials from the National Museum of Kazakhstan, the project explores political, economic, legal, and cultural changes that reshaped traditional nomadic society.

**Academic Focus:**
- Russian colonial expansion into Central Asia
- Administrative reforms and their social impact
- Transformation of traditional Kazakh governance
- Economic disruption of nomadic pastoralism
- Cultural and spiritual consequences
- Resistance and adaptation strategies

## üöÄ Installation

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - runs entirely in the browser!

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/colonial-kazakhstan.git
   cd colonial-kazakhstan
   ```

2. **Open the website**
   ```bash
   # Simply open index.html in your browser
   open index.html
   # Or on Windows
   start index.html
   # Or on Linux
   xdg-open index.html
   ```

3. **That's it!** No build process required.

### Optional: Local Server

For better performance with modern browsers, you can use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx serve

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.
